Appendix 5: Derivational Suffix and Affix Inventory

Consonant mutations before selected suffixes: k/c→č, g→ž, h→š (before -ji, -ka, -ko, -nik, -ny, -ok, -sky, -stvo). Soft nj hardens before -nik and -ny. Suffixes starting with -o- change to -e- after soft consonants.

Noun-Forming Suffixes​

Noun → Noun​

SuffixSemantic classOutput genderExample
-okdiminutive (m., hard stem)m.pes → pesok "doggy"
-ik / -ekdiminutive (m., soft stem)m.muž → mužik "little man"
-kadiminutive (f.)f.kniga → knižka "booklet"
-kodiminutive (n.)n.polje → poljko "small field"
-stvoabstract quality from person/thingn.prijatelj → prijateljstvo "friendship"; muž → mužstvo "manhood"
-iščeplace associated with nounn.voz → vozišče "parking lot"; ogonj → ognišče "hearth"
-je (+ prě-/pri-/pod- etc.)place in relation to another placen.morje → primorje "land by the sea"; brjuh → podbrjušje "underbelly"
-nikperson in charge of / handler of; containerm.glava → glavnik "boss"; kniga → knižnik "bookcase"
-nicafemale equivalent of -nikf.rada → radnica "female counsellor"
-ecperson representing quality; inhabitantm.Albanija → Albanec "Albanian"; glupy → glupec "fool"
-icafemale equivalent of -ecf.Albanec → Albanica "Albanian woman"
-ka (person)female equivalent (general)f.prijatelj → prijateljka "girlfriend"; gradžanin → gradžanka "female citizen"
-an(in)inhabitant (alternative to -ec)m.(cf. Slovjanin, Moskvjanin)
-e (-et- stem)young animaln.kot → kote "kitten"; jelenj → jelene "fawn"

Adjective → Noun​

SuffixSemantic classOutput genderExample
-ostabstract quality from adjectivef.dobry → dobrost "goodness"
-ecperson embodying qualitym.izgnany → izgnanec "exile"
-iščeplace with qualityn.mokry → mokrišče "wetland"

Verb → Noun​

SuffixSemantic classOutput genderExample
-teljagent/professional (one who does)m.učiti → učitelj "teacher"
-ačprofessional/tradem.kovati → kovač "blacksmith"; krojiti → krojač "tailor"
-arprofessional/tradem.pekti → pekar "baker"
-iščeplace of activity (often from L-participle)n.igrati → igrališče "playground"; žiti → žilišče "dwelling"
-njeverbal noun (gerund), from past pass. part.n.dělati → dělanje "doing"; žiti → žitje "life"
-tjeverbal noun (alternative after -t stem)n.piti → pitje "drinking"

Adjective-Forming Suffixes​

Noun → Adjective​

SuffixSemantic classExample
-skyrelating to geography, entity, professionžena → žensky "female"; morje → morsky "marine"
-jirelating to animals, divine entities, personsčlověk → člověčji "human"; konj → konji "horse's"
-nygeneral relational adjectivenoč → nočny "nocturnal"; pismo → pisemny "written"
-ěnymade of (material)drěvo → drěvěny "wooden"; steklo → steklěny "of glass"
-ovy / -evyhaving characteristics of; colour/typepomaranča → pomarančevy "orange (colour)"
-ovity / -evityresembling, -likemalpa → malpovity "ape-like"

Adjective → Adjective​

Suffix/PrefixSemantic classExample
ne- (prefix)negationdobry → nedobry "not good"
prě- (prefix)intensification ("very/too")umorjeny → prěumorjeny "exhausted"
-ovityweakening/approximation ("-ish")zeleny → zelenovity "greenish"
bez- … -ny (circumfix)absence of qualitydom → bezdomny "homeless"; naděja → beznadějny "hopeless"
-šnjifrom adverb: adjectivising temporal/local adverbsvčera → včerašnji "yesterday's"; dnes → dnešnji "today's"

Comparative and Superlative​

FormFormationExample
Analytic comparativevyše / bolje + positivevyše prosty "simpler"
Synthetic comparative (adj.)-ějši / -ejši (soft) replacing -y/-ibogaty → bogatějši
Synthetic comparative (adj. in -ky)-šikratky → kratši; vysoky → vysši
Comparative adverb-ěje / -ejenovo → nověje
Comparative adverb (from -ky)-je (+ iotation)daleko → dalje; široko → širje
Superlativenaj- + comparativenajnověje, najvysši
Superlative (simplified)naj- + positivenajnovy

Verb → Adjective​

SuffixSemantic classExample
-livytendency/habitgovoriti → govorlivy "talkative"

Verb-Forming Suffixes and Prefixes​

Noun/Adjective → Verb​

Suffix/PatternSemantic classExample
-ovatigeneral denominal verb ("act like; do with")malpa → malpovati "act like a monkey"; noč → nočevati "spend the night"
o- + -itigive/acquire quality (noun/adj. → verb)kamenj → okameniti "stone"; světlo → osvětliti "illuminate"; svoboda → osvoboditi "free"
obez- + -itideprive of quality (noun → verb)glava → obezglaviti "decapitate"; dom → obezdomiti "make homeless"
-ětibecome (adj. → verb of becoming)běly → bělěti "turn white"; bogaty → bogatěti "grow rich"
o- + -iti (from adj.)make (someone) have the characteristicčrny → očrniti "blacken"
u- + -iti (from comparative)make more/betterlěpši → ulěpšiti "improve"; menši → umenšiti "reduce"
od- + -iti (from adj.)remove the characteristicčrny → odčrniti "remove black colour"

Adverb → Adjective​

SuffixExample
-šnjinyně → nyněšnji "current, present"

Adjective → Adverb​

PatternExample
Replace adj. ending with -odobry → dobro "well"; svěži → svěže "freshly"
Replace comparative -(ěj)ši with -(ě)jeslabějši → slaběje "more weakly"

Compound Word Formation​

Two roots joined by connecting vowel -o- (hard) / -e- (soft):

ExampleComponentsMeaning
vodopadvoda + padatiwaterfall
samokritikasam + kritikaself-criticism
myšelovkamyš + lovitimousetrap
zemjetresenjezemja + tresenjeearthquake

Borrowings from English may use hyphen: rok-muzika "rock music", veb-stranica "web page".

Negative and Intensifying Prefixes (Non-verbal)​

PrefixFunctionExample
ne-negation (adj., nouns)neprijetny "unpleasant", nedobry "not good"
prě-intensification, excessprěumorjeny "exhausted", prěsladky "too sweet"
naj-superlativenajdobry "best", najnověje "most recently"
bez-absence (with -ny or -nik)bezdomny "homeless", bezbožnik "godless person"
malo-little, small (compounding)maloznačny "insignificant"
mnogo-much, many (compounding)mnogokratno "many times"
samo-self- (compounding)samokritika "self-criticism"
su-co-, together (compounding)surabotati "co-work"
