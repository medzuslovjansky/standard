# Develop

Tools, APIs, and ways to contribute to the Interslavic ecosystem.

## Tools

- [Dictionary](https://interslavic-dictionary.com) — the Interslavic dictionary
- Transliterator — convert between Latin and Cyrillic scripts
- Declinator — noun and adjective declension tables
- Conjugator — verb conjugation tables
- Keyboards — input methods for Interslavic characters
- Spellcheck — spelling and grammar checking

## For Developers

- Dictionary API
- Transliteration libraries
- NLP resources
- GitHub repositories: [github.com/medzuslovjansky](https://github.com/medzuslovjansky)

## Contribute

- **Code** — Contribute to tools and infrastructure on [GitHub](https://github.com/medzuslovjansky)
- **Standard** — Help improve the [language specification](/standard)
- **Translate** — Help translate resources into other languages, or translate content into Interslavic
- **Community** — Moderate forums, welcome newcomers, organize events
